# Scaffolds

The `app.scaffolds` object looks something like this:

```js
var scaffolds = {
  component: {
    views: {
      scripts: {},
      styles: {},
      templates: {}
    }
  },
  blog: {
    views: {
      scripts: {},
      styles: {},
      post: {},
      index: {},
      list: {}
    }
  }
};
```
